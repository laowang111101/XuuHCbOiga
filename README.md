## 前言

随着我国城市化进程的加速，人们对于公寓出租的需求越来越大。为了解决这一问题，我们开发了一款基于SpringBoot的公寓出租系统，旨在为广大用户提供便捷、高效的公寓租赁服务。本系统采用Java作为后端开发语言，结合Spring Boot框架，确保了服务端应用的高效性和稳定性。前端则利用Vue.js技术，为用户提供了直观、易用的交互界面。系统服务于管理员和用户两种角色，提供了全面的服务和管理功能。

## 内容介绍

公寓出租系统是一款基于Java和Spring Boot框架开发的应用程序，其主要功能包括房源信息浏览、预约看房、租赁订单管理、个人信息管理等。系统采用前后端分离的设计模式，后端采用Java语言，基于Spring Boot框架进行开发，确保了系统的高效性和稳定性。前端则采用了Vue.js技术，为用户提供了直观、易用的交互界面。
